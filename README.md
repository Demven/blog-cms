# blog-cms

## Project setup
```
nvm use 14.7
npm install
npm run dev
```

Run API locally and then open http://cms.dmitry-salnikov.info:8080

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
